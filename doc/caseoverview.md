Mastercard case
Case Details

Participants will build a chatbot that can:
Parse natural language prompts.
Translate them into SQL or equivalent queries.
Connect securely to a database (e.g., PostgreSQL, MySQL, Snowflake).
Return results in a user-friendly format (text, table, chart).
Handle ambiguous or incomplete queries gracefully.
Log interactions for audit and improvement.
Example Prompts:

"Total transactions for S1Ik Pay in Q1 2024."
"Top 5 merchants by revenue in Kazakhstan last year."
"Decline rate for CID 12345 in October."

Technical Requirements
Core Capabilities:

NLP Engine: Intent recognition and entity extraction.
Query Translator: Converts parsed intent into SQL.
Database Connector: Secure access to structured data.
Response Formatter: Text, table, or chart output.
Error Handling: Graceful fallback for invalid or unclear queries.
